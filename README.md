# VaultNotes

## Overview
VaultNotes is a secure backend API for a note management application.  
This project was built using Node.js and Express and follows RESTful API design principles.  
It serves as the backend foundation for a scalable full-stack notes platform.

---

## Tech Stack
- Node.js
- Express.js
- JavaScript
- REST API Architecture

---

## Features
- Express server configuration
- Middleware setup (JSON parsing, CORS)
- RESTful routing structure
- Local development environment on port 5000
- Scalable backend project structure
